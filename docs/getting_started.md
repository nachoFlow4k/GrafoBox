# Getting started

1. Clona este repositorio. 
2. `pip install -r requirements.txt` (si se va a cambiar la docu, sino, no hace falta)
3. `npm init`
4. `npm install --save-dev jest`


## Correr los tests

+ Para correr los tests, es necesairo ir hasta la carpeta tests.
    En esta, con `npm test` corre todos los tests.

+ Si se quiere debugear, en vs code, con ir a `run and debug` y de ahí se debería poder correr sin problemas
